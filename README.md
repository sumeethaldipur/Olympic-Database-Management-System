# Olympic Database Management System

We developed a Django-based web application to manage and host a large-scale international sporting events such as the Olympics. The platform enabled organizing committees to publish competition details and provided spectators with interactive features including reactions and comments.

### This is a group project by Abhishek Chopra, Sumeet Haldipur, and Yash Brid for Sardar Patel Institute of Technology
